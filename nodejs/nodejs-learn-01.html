    # 2、events模块

    通过events来绑定和监听事件。

    event：事件

    handler：事件处理程序

    通过**on(event,handler)**来绑定事件及事件处理程序

    通过**emit(event)**来触发事件，然后事件处理程序执行任务

    var events = require('events'); // 引入event模块
var eventEmitter = new events.EventEmitter(); // 创建EventEmitter对象
var connectHandler = function connected(){// 事件处理程序
console.log('连接成功。');
eventEmitter.emit('data_received');//触发data_received事件
}
eventEmitter.on('connection',connectHandler); // 绑定connection事件处理程序
eventEmitter.on('data_received',function(){ // 使用匿名函数绑定data_received事件
console.log('数据接收成功！');
});
eventEmitter.emit('connection'); // 触发connection事件
console.log("程序接收完毕！");

    # 3、Buffer

    缓冲区

    buf1：缓冲区1，buf2：缓冲区2

    write('ldg')：写入

    toString('ldg')：从缓冲区读取数据

    JSON.stringfy(Buffer.from([0x1,0x2]))：字符串转换成Buffer

    concat([buf1,buf2])：缓冲区合并

    buf1.compare(buf2)：缓冲区比较

    buf2.copy(buf1,1)：缓冲区拷贝 //将buf2插入到buf1指定位置上，同时会覆盖buf1原先的值

    buf1.slice(0,2)：缓冲区裁剪 // 按指定位置裁剪缓冲区

    # 4、文件流

    createReadStream：可读流

    createWriteStream：可写流

    setEncoding('utf8')：设置编码格式

    write(data,'utf8')：按指定编码格式向可写流中写入数据

    end()：写入完成后，需要添加end表示写入完成

    readerStream.pipe(writeStream)：管道操作，将可读流数据通过管道操作，写入到可写流数据中

    # 5、exports模块
